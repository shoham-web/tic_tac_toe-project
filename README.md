## Project Structure
Board class
- Manages the game board
- Checks win and tie conditions

Player class
- Handles player input
- Manages player symbol

PlayComputer class
- Inherits from Player
- Generates random valid moves

Game class
- Controls the game loop
- Handles game modes and turn switching
